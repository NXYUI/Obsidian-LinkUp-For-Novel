# Obsidian LinkUp For Novel

Obsidian LinkUp For Novel is a desktop workspace for managing novel-writing projects backed by an Obsidian vault.

It brings planning, character and setting sheets, episode management, consistency checks, writing statistics, backups, and an AI-assisted document agent into one local application.

## Download

Installers are distributed through the GitHub Releases page.

## Notes

- The app works with local vault folders.
- Project data and app settings are stored locally on the user's machine.
- Release artifacts are uploaded separately from this repository's source files.
