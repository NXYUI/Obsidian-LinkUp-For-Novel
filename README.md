# Obsidian LinkUp For Novel

Obsidian LinkUp For Novel은 소설 집필 과정에서 흩어지기 쉬운 자료를 하나의 Obsidian 볼트 안에서 정리하고, 그 자료를 더 쉽게 읽고 연결하고 점검하기 위한 데스크톱 앱입니다.

이 앱은 원고를 대신 써주는 도구라기보다, 작가가 이미 만들어 둔 세계관, 인물, 설정, 플롯, 회차 메모를 작업 가능한 형태로 다시 보여주는 보조 작업대에 가깝습니다. Obsidian의 자유로운 문서 구조는 그대로 유지하면서, 장편 작업에서 반복적으로 필요한 정리, 확인, 이동, 백업 과정을 줄이는 것을 목표로 합니다.

특히 문서가 많아질수록 생기는 문제에 초점을 둡니다. 어떤 설정이 어디에 있었는지 찾기 어렵거나, 캐릭터와 사건의 연결이 흐려지거나, 이전에 적어 둔 내용과 새로 쓴 내용이 어긋나는 상황을 줄이기 위해 만들어졌습니다.

모든 작업은 사용자의 로컬 파일과 로컬 앱 데이터를 기준으로 이루어집니다. 배포 파일은 GitHub Releases와 [아카라이브 채널](https://arca.live/b/aiofn?category=update)에서 제공합니다.
